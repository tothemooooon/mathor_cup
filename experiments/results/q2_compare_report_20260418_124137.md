- 分组：

## 3. 关键发现

1. **Q2 回退后的主线（直入模 + 自适应 λ）已稳定可复现运行**，且输出了 `selected_lambda / adaptive_trace / gap_to_dp` 证据链。
2. 相比固定 λ，**自适应 λ 的 best 有提升**：
   - `214648 -> 207427`，下降 `3.36%`。
3. 但在 3 次重复下，**自适应 λ 的 median 与固定 λ 持平**（均为 `214648`），说明本轮预算下改进主要体现在“偶发命中更优解”，稳定性尚不足。
4. 分支切割结果显著优于 QUBO 两组：
   - `97920`（gap_to_dp=`1.1640`），已明显接近 DP 基准。
5. 现阶段主矛盾不是“Q2 公式是否能表达时间窗”，而是 **QUBO+SA 在该实例上的搜索效率仍偏弱**。

## 4. 结论

- 你要求的回退改造目标已经完成：Q2 不再走 B+C 迭代，正式主线改为“直入模 + 自适应 λ”。
- 从质量对照看：
  - **DP（金标准）**：`84121`
  - **分支切割**：`97920`（较 DP 高 `16.40%`）
  - **QUBO 自适应（best）**：`207427`（较 DP 高 `146.58%`）
- 因此，若论文强调“求解质量上界与可解释对照”，建议把 **DP + 分支切割**作为质量锚点，把 **QUBO 自适应**定位为可扩展近似框架，并如实给出当前 gap。
